# AI Text Detector (Unicode Checker)

Ein einfaches JavaScript-Tool zur Erkennung von versteckten oder auffälligen Unicode-Zeichen in Texten.

## 🔍 Überblick

Dieses Projekt analysiert eingegebenen Text und erkennt unsichtbare oder ungewöhnliche Unicode-Zeichen, die häufig in manipulierten oder automatisch generierten Texten vorkommen.

Ziel ist es, einfache Textmanipulationen sichtbar zu machen und Inhalte auf verdächtige Zeichen zu prüfen.

## ⚙️ Funktionsweise

Das Tool prüft den eingegebenen Text auf eine Liste von Unicode-Zeichen, darunter:

- Zero Width Spaces (`\u200B`)
- Zero Width Joiner / Non-Joiner (`\u200C`, `\u200D`)
- No-Break Spaces (`\u00A0`, `\u202F`)
- Bidirektionale Steuerzeichen (LTR / RTL)
- Unsichtbare Formatierungszeichen
- Veraltete Unicode-Steuerzeichen

Wird eines dieser Zeichen gefunden, wird der Text als:

- ❌ **"This Text is AI Generated"**

markiert, andernfalls als:

- ✅ **"This Text is generated by a Human"**
